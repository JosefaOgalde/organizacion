## Traer tu carpeta de Windows

1. Abre: `organizacion\index\clientes\MKOF\MOVA\auditoria-charlas\`
2. Copia **todo** el contenido de `MOVA-Auditoria-Charlas` ahí.
3. En Cursor: abre la carpeta **`organizacion`** (raíz del repo).
