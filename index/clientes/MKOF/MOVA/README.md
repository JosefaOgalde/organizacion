# MOVA · Auditoría Charlas (proyecto MKOF)

Subproyecto de **MKOF - Talk** dentro del organizador.

## Ver en el navegador

| Página | URL (con `SERVIR.bat` o `npx serve .`) |
|--------|----------------------------------------|
| Ficha MKOF | `http://localhost:3000/index/clientes/MKOF.html` |
| Proyecto MOVA | `http://localhost:3000/index/clientes/MKOF/MOVA.html` |
| Código charlas | `http://localhost:3000/index/clientes/MKOF/MOVA/auditoria-charlas/` |

## Traer tu carpeta de Windows

1. Abre: `organizacion\index\clientes\MKOF\MOVA\auditoria-charlas\`
2. Copia **todo** el contenido de `MOVA-Auditoria-Charlas` ahí.
3. En Cursor: abre la carpeta **`organizacion`** (raíz del repo).
