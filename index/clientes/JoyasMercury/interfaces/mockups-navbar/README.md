# Navbar · maqueta Joyas Mercury

Maquetas HTML/CSS del navbar desktop y **menú hamburguesa móvil** (Fase 2).

## Archivos

| Archivo | Descripción |
|---------|-------------|
| `navbar-jm-maqueta.html` | Navbar desktop |
| `navbar-jm-maqueta.css` | Estilos navbar desktop |
| `menu-hamburguesa-jm-maqueta.html` | **Menú hamburguesa móvil · drawer propuesto** |
| `menu-hamburguesa-jm-maqueta.css` | Estilos drawer |
| `navbar-jm-maqueta-desktop.png` | Captura navbar desktop |
| `menu-hamburguesa-jm-maqueta-open.png` | Captura drawer abierto |

## Ver en navegador

**Navbar desktop:**
```
http://localhost:3000/index/clientes/JoyasMercury/interfaces/mockups-navbar/navbar-jm-maqueta.html
```

**Menú hamburguesa (interactivo · drawer abierto con `?open=1`):**
```
http://localhost:3000/index/clientes/JoyasMercury/interfaces/mockups-navbar/menu-hamburguesa-jm-maqueta.html?open=1
```

## Generar PNG

```bash
python3 scripts/capturar-jm-navbar-maqueta.py
python3 scripts/capturar-jm-menu-hamburguesa.py
```
