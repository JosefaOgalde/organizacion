# Navbar · maqueta Joyas Mercury (desktop)

Maqueta HTML/CSS del navbar **solo desktop** con identidad del manual de marca.

## Archivos

| Archivo | Descripción |
|---------|-------------|
| `navbar-jm-maqueta.html` | Maqueta desktop |
| `navbar-jm-maqueta.css` | Estilos |
| `navbar-jm-maqueta-desktop.png` | Captura desktop |

## Ver en navegador

```
http://localhost:3000/index/clientes/JoyasMercury/interfaces/mockups-navbar/navbar-jm-maqueta.html
```

## En la landing JM

Sección **Mockups navbar** debajo de Landings referencia, con botón **Visualizar maqueta interactiva**.

## Generar PNG

```bash
python3 scripts/capturar-jm-navbar-maqueta.py
```

## Identidad JM

- Gold `#ECC54A` / `#A97E23`
- Rosa `#C88F9C` · Nude `#D8BFB1` · Gris `#C4C4C4`
