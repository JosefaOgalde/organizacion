# Maqueta diseño · Inicio · Joyas Mercury

Landing desktop **idéntica a la referencia entregada**: hero con joyería, barra de valor, colecciones, novedades, últimas unidades y footer dorado.

## Estructura

```
maqueta-diseno/
├── index.html              ← página principal (abrir directo o con Vite)
├── default_shadcn_theme.css
├── package.json
├── vite.config.ts
├── src/
│   ├── main.ts
│   └── styles.css
└── guidelines/
```

## Ver sin instalar

```
http://localhost:3000/index/clientes/JoyasMercury/interfaces/maqueta-diseno/index.html
```

(con `node scripts/organizacion-server.js`)

## Desarrollo con Vite

```bash
cd index/clientes/JoyasMercury/interfaces/maqueta-diseno
pnpm install   # o npm install
pnpm dev
```
