# Guidelines · Maqueta diseño Joyas Mercury

## Objetivo

Replicar fielmente la captura de referencia del **Inicio desktop** para validación con Camila antes de implementar en WordPress/Astra.

## Secciones (orden)

1. Header — isotipo + nav Inicio/Colecciones
2. Hero — foto joyería + copy centrado
3. Trust bar — 4 beneficios
4. Colecciones — fondo floral, 3 círculos
5. Novedades — grid 4 productos + CTA
6. Últimas unidades — grid 4 con badges stock
7. Footer — gold, columnas, WhatsApp

## No incluir en esta maqueta

- Carrusel hero múltiple
- Menú hamburguesa (solo desktop)
- Variantes footer A/D/E (solo footer gold de referencia)

Colocar PNG en `../../identidad/` y capturas sitio en `../../flujo-actual/` para hero y círculos reales.
