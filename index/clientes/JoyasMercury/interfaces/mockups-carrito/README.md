# Banner carrito vacío — Joyas Mercury

Banner **1000 × 500 px** para la sección «Mi Carrito» vacío, según mockup de referencia.

## Archivos

| Archivo | Uso |
|---------|-----|
| `banner-carrito-vacio.html` | Solo el banner (1000×500) |
| `jm-banner-carrito-vacio-1000x500.png` | PNG exportado |
| `wireframe-carrito-vacio.html` | Página completa: banner + Favoritos Mercury |

## Ver en local

```
http://localhost:3000/index/clientes/JoyasMercury/interfaces/mockups-carrito/banner-carrito-vacio.html
http://localhost:3000/index/clientes/JoyasMercury/interfaces/mockups-carrito/wireframe-carrito-vacio.html
```

## Regenerar PNG

```bash
python3 scripts/capturar-jm-banner-carrito.py
```

## Contenido del banner

- Izquierda: icono bolsa + corazón, título «Tu carrito está **vacío**» (vacío en rosa), separador dorado, texto, CTA «Ver colecciones»
- Derecha: foto lifestyle joyas con degradado crema
- Medidas exactas: **1000 × 500 px**
