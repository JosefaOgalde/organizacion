# Banner Mi Carrito — Joyas Mercury

Banner **general** **1000 × 500 px** para la sección «Mi Carrito» (con o sin productos).

## Archivos

| Archivo | Uso |
|---------|-----|
| `banner-carrito-vacio.html` | Preview del banner (nombre legacy) |
| `jm-banner-carrito-1000x500.png` | PNG principal para WP/Elementor |
| `wireframe-carrito-vacio.html` | Página completa con banner + favoritos |

## Contenido del banner

**Izquierda**
- Eyebrow «Joyas Mercury»
- Título «Mi **Carrito**» (Carrito en rosa)
- Texto general sobre revisar selección
- Confianza: envíos · garantía · compra segura
- CTA «Seguir comprando»

**Derecha**
- Foto lifestyle joyas + ilustración bolsa de compras dorada con joyas alrededor

> No menciona carrito vacío — sirve para cualquier estado del carrito.
