# Mova · Auditoría Charlas

Cliente y proyecto **MOVA-Auditoria-Charlas** integrados en el organizador.

## Ver en el navegador

| Página | URL (con `npx serve .`) |
|--------|-------------------------|
| Listado clientes | `http://localhost:3000/index/clientes/` |
| Ficha Mova | `http://localhost:3000/index/clientes/Mova.html` |
| Hub proyecto | `http://localhost:3000/index/clientes/mova/` |
| Código charlas | `http://localhost:3000/index/clientes/mova/auditoria-charlas/` |

## Traer tu carpeta de Windows

1. Abre en el explorador: `organizacion\index\clientes\mova\auditoria-charlas\`
2. Copia **todo** el contenido de `MOVA-Auditoria-Charlas` ahí (o mueve la carpeta entera).
3. En Cursor: **File → Open Folder** → carpeta `organizacion` (raíz del repo).
4. Si tenías reglas en `.cursor/rules/` del proyecto viejo, cópialas a `organizacion/.cursor/rules/mova.mdc` (o fusiona el contenido).

Alternativa en la raíz del repo: `organizacion/MOVA-Auditoria-Charlas/` — el agente también la reconoce.
