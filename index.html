Información que necesito por cliente

Para ayudarte a planificar tu semana, necesito estos datos de cada cliente o proyecto:

  1. Nombre del cliente o proyecto
  2. Tipo: freelance, full-time o personal
  3. Descripción breve del trabajo
  4. Fecha límite o entrega final
  5. Entregables principales (qué hay que entregar)
  6. Horas semanales que puedes dedicarle
  7. Prioridad (alta / media / baja)
  8. Notas: reuniones fijas, horarios, contactos
